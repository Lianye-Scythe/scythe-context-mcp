# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Expected Behavior

- Be respectful and constructive.
- Keep technical disagreement focused on the work.
- Do not post API keys, proxy tokens, private source code, local index databases, or other sensitive material.

## Enforcement

Report unacceptable behavior privately to the project maintainer when possible. Security-sensitive reports should follow [SECURITY.md](SECURITY.md).
